Sforte Microfinance Admin is a specialized financial management tool designed to bridge the gap between local market stakeholders and digital banking. The software serves as a centralized dashboard for managing member data, tracking investments, and issuing loans. By utilizing a relational database structure, the application ensures data integrity across multiple financial tables, allowing for complex reporting such as liquidity aggregates and member portfolios.

The purpose of creating this software is to provide a scalable, secure, and automated system for a startup microfinance institution in Abuja. The goal was to replace manual record-keeping with a digital solution capable of:

Performing real-time financial calculations (Aggregates).

Maintaining strict relationships between members and their capital (Relational Mapping).

Filtering sensitive financial data by date and status to assist in regulatory compliance

To develop this software, I utilized a modern full-stack environment focused on speed and database reliability:Language: JavaScript (Node.js)         Backend Framework: Express.js for API routing and server management.Database: PostgreSQL (Hosted on Render) for robust relational data storage.                       Frontend: HTML5, CSS3, and Vanilla JavaScript for a fast, responsive user interface.                             Version Control & Deployment: Git/GitHub for code management and Render for cloud hosting.                 Development Tools: Visual Studio Code (VS Code) and Postman for API testing.
